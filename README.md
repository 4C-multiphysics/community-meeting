# 4C Community Meeting


The 4C Community Meeting is a forum for open exchange and discussion among all stakeholders in the 4C community. The meeting is open to the public. **Everybody** is enouraged to add items to the agenda.

The community meeting regularly occurs on the **last Wednesday of each month at 4:15pm CET**.

## Purpose of this repository

This repository is used for the organization and documentation of 4C Community Meetings. It does not contain code apart from templates for meeting issues and other documents for organization purposes.

## Meeting organization

Each meeting is organized in an issue within this GitHub project.

- To organize an instance of the meeting, please open a new issue and fill the "Meeting" issue template with all relevant information. Each meeting will be oragnized by a _host_, who is responsible for scheduling the meeting, curating the agenda, and moderating the meeting (or cancelling the meeting if there are no items on the agenda two days before the meteing).
- **Everybody** is enouraged to add items to the agenda.
- Each meeting's outcome is docuemnted in meeting notes by a _note taker_.
- The roles of _host_ and _note taker_ rotate, with the _host_ of the last meeting usually becoming the _note taker_ for the next meeting.
